# Location St-Flo 🌊

Site vitrine moderne pour **Location St-Flo** — location de quads et de bateaux
(avec ou sans permis) à Saint-Florent, Haute-Corse.

Refonte au goût du jour reprenant les informations du site existant
(activités, tarifs, lieux à explorer, contact).

## ✨ Caractéristiques

- Design responsive moderne (mobile / tablette / desktop)
- Palette méditerranéenne, typographies *Fraunces* + *Manrope*
- Hero immersif, animations au défilement, navigation collante
- Grille de tarifs avec bascule **basse / haute saison**
- Formulaire de contact (ouverture de la messagerie pré-remplie)
- Aucune dépendance, aucun build — HTML / CSS / JS pur

## 🗂 Structure

```
index.html        Page unique
css/styles.css    Styles
js/script.js      Interactions (nav, reveal, tarifs, formulaire)
assets/img/       Photos (reprises du site location-stflo.com)
```

## 🚀 Lancer en local

Ouvrez simplement `index.html` dans un navigateur, ou servez le dossier :

```bash
python3 -m http.server 8000
# puis http://localhost:8000
```
